## Shared Decisions

- `.NET 10 SDK 10.0.203` is installed locally; use `dotnet new sln --format sln` because the SDK defaults to the new `.slnx` XML format which the task spec does not accept.
- Per-project csproj files intentionally omit `<TargetFramework>` — it is set once in `Directory.Build.props` per ADR-005. Adding it back in a csproj will duplicate and may shadow the central setting.
